Software Inc. Mod Maker

Download Version 1.0 Intel Download Version 1.1 Intel Download Version 1.1 ARM32 and ARM64 Download Version 1.2 Intel Download Version 1.2 ARM64

Setup: Extract the zip folder downloaded

If not done already; install the .NET Framework

Open the Setup.exe file within the extracted folder

Create a new folder called whatever you want to call your mod

Create a folder in your mod folder that you created called "SoftwareTypes" (case sensitive)

Create a folder in your mod folder that you created called "CompanyTypes" (case sensitive)

Create a folder in your mod folder that you created called "NameGenerators" (case sensitive)

Your setup is now complete. Go on and now mod the game.
